## Download

1. Download `MARCY.zxp` from [Releases](https://github.com/Marcycuteaf/ae-marcy-toolkit/releases/latest)
2. Install with [ZXP Installer](https://aescripts.com/learn/zxp-installer/)
3. Restart After Effects
4. **Window → Extensions → MARCY**

---

## Features

- **Text & FX** — EN / JP / SUB · Fill · Grad · Box · Clean
- **Align & Anchor** — align / distribute · 3×3 anchor · paragraph L/C/R
- **Tools** — Pre-C · Null · Adj · Sep XY · Solid · Cam · Light · Crop · In `[` / Out `]`

Options: Color · Qty · separate pre-compose · single / merge
