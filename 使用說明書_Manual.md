# MARCY AE Toolkit · 使用說明書

After Effects CEP 面板 — 文字、效果、對齊錨點、圖層工具、Auto Crop、時間軸 In/Out

---

## 安裝

1. 從 [Releases](https://github.com/Marcycuteaf/ae-marcy-toolkit/releases/latest) 下載 `MARCY.zxp`
2. 用 ZXP Installer 安裝
3. 重啟 AE → **Window ▸ Extensions ▸ MARCY**

---

## 介面

### Text & FX
| 按鈕 | 功能 |
| --- | --- |
| EN / JP / SUB | 建立文字預設 |
| Fill / Grad / Box / Clean | 文字效果 |

### Align & Anchor
- 對齊選取 / 對齊合成
- 左中右、上中下、H/V 分佈
- Anchor TL~BR、Paragraph L/C/R（3D 可用）

### Tools
| 按鈕 | 功能 |
| --- | --- |
| Pre-C | 預合成 |
| Null | 建立 Null 並 parent |
| Adj | 調整圖層 |
| Sep XY | 分離 Position |
| Solid / Cam / Light | 建立圖層 |
| Crop | Auto Crop |
| In [ / Out ] | 進出點對齊時間 |

### 選項
- **Color** — 圖層標籤色
- **Qty** — 數量
- **分別預合成** — 逐個或合併預合成
- **單一/合併** — 多選共用一個 Null/Adj/Solid，或各一個
